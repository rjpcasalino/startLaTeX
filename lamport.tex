What is a \verb|documentclass| anyway? Well, there are some standard classes such as: \verb|article|, \verb|report|, \verb|book|, \verb|letter|, and \verb|slides|.

The \verb|article| class is generally used for shorter documents and \verb|report| for longer (p.19).

``Convention' dictates that punctuation go inside the quotes, like ``this,'' but [Lamport] think[s] it's better to do ``this''.

Lamport does not mention the use of Backtick (`) to get quotes right. Or maybe he does and we missed it. In any case,

\begin{verbatim}
``this'' is how you'd want to write/typeset quotes so they are produced correctly.
\end{verbatim}

If you would like to know more: \url{https://stuff.mit.edu/afs/sipb/project/www/latex/guide/node12.html}. We'll hammer this home in a few milliseconds.
